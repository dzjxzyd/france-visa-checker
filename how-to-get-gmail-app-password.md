# How to Get a Gmail App Password

## What is an App Password?

An App Password is a **16-digit password** generated by Google that allows third-party apps (like Python scripts) to send emails via SMTP. It replaces your Gmail account password for added security.

---

## Prerequisites

Your Google account must have **2-Step Verification** enabled.

---

## Steps

### Step 1: Enable 2-Step Verification

1. Open your browser and sign in to your Google account
2. Go to [Google Account Security Settings](https://myaccount.google.com/security)
3. Find the **"How you sign in to Google"** section
4. Click **"2-Step Verification"**
5. Follow the prompts to complete setup (usually requires a phone number)

### Step 2: Generate an App Password

1. Go to the [App Passwords page](https://myaccount.google.com/apppasswords)
   - Or search for **"App passwords"** in your Google Account security settings
2. Enter a name in the **"App name"** field (e.g., `Python Email`)
3. Click **"Create"**
4. Google will generate a **16-digit password** (e.g., `abcd efgh ijkl mnop`)
5. **Copy and save this password** — you won't be able to see it again!

### Step 3: Use It in Your Code

Paste the 16-digit password into your Python script:

```python
APP_PASSWORD = "abcd efgh ijkl mnop"
```

---

## Important Notes

- ⚠️ **Do NOT use your Gmail login password** — you must use an App Password
- 🔒 **Do NOT commit your App Password to GitHub** or any public repository
- 🗑️ You can revoke it anytime at the [App Passwords page](https://myaccount.google.com/apppasswords)
- 📌 Each App Password only needs to be generated once and can be reused
